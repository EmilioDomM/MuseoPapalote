# Museo Papalote (Android App)

**EN:** An Android app made to help *Museo Papalote* attract more visitors.
This app provides an **interactive map** so visitors can guide themselves and have a more immersive experience.

**ES:** Una app Android hecha para ayudar al *Museo Papalote* a recibir más visitantes.
Esta app ofrece un **mapa interactivo** para que los visitantes se guíen y vivan una experiencia más inmersiva.

---

## Features / Funciones

- **EN:** Interactive map for self‑guided visits inside the museum  
  **ES:** Mapa interactivo para recorridos autoguiados dentro del museo
- **EN:** Clear navigation between sections/rooms  
  **ES:** Navegación clara entre secciones/salas
- **EN:** Designed for a simple, friendly visitor experience  
  **ES:** Diseñada para una experiencia sencilla y amigable

---

## Tech / Tecnología

- **EN/ES:** Kotlin
- **EN/ES:** Jetpack Compose (Material 3)
- **EN/ES:** Android Studio + Gradle

## Getting Started / Empezar

### Requirements / Requisitos
- **EN:** Android Studio (recommended: latest stable)  
  **ES:** Android Studio (recomendado: última versión estable)

### Run / Ejecutar
1. **EN:** Clone the repo  
   **ES:** Clona el repositorio
   ```bash
   git clone https://github.com/EmilioDomM/MuseoPapalote.git
   cd MuseoPapalote
   ```
2. **EN:** Open in Android Studio and sync Gradle  
   **ES:** Abre en Android Studio y sincroniza Gradle
3. **EN:** Run the `app` configuration on an emulator/device  
   **ES:** Ejecuta `app` en un emulador/dispositivo
